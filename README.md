# SharkIsland

## Implemented domains:

### Group A - Cubemaps
### Group B - HDR/Bloom effects


## Controls:

- 'WASD' - movement
- 'V' - enable/disable movement
- 'B' - turn on/off the binoculars
- 'C' - lock/unlock chest
- 'L' - turn on/off the lamp

- 'P' - disable bloom effects
